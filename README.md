
# Rock Paper Scissors Game 

## Description
A simple and interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.  
The game allows users to play against the computer, with instant results displayed after each round.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- User vs Computer gameplay
- Random computer move generation
- Instant result display (Win / Lose / Draw)
- Clean and responsive user interface
- Beginner-friendly project structure

## How to Play
1. Choose Rock, Paper, or Scissors.
2. The computer will randomly select its move.
3. The winner is decided based on standard Rock Paper Scissors rules.
4. The result is displayed immediately on the screen.
